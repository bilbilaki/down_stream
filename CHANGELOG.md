## 0.0.1

* Initial release of DownStream
* Core Dart proxy server for streaming and caching
* Sparse file storage with efficient range tracking
* Support for custom User-Agent
* HTTP and SOCKS5 proxy configuration
* Download cancellation support
* Real-time progress tracking
* File preview with MIME type detection
* Collections management API
* Export and cleanup functionality
* Debounced metadata saves
* Bitmap optimization for large files (>100MB)
